# Heart Medical Assistant (RAG + Ollama)

A local AI-powered **Heart Medical Assistant** built using **Django**, **FAISS**, **Sentence Transformers**, and **Ollama (Mistral)**.

The assistant provides **educational heart health information** related to:
- Heart symptoms
- Blood pressure
- Exercise guidelines
- Heart-healthy diet
- Medical tests
- Surgery information
